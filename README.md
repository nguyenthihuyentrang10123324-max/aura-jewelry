# Aura Jewelry - Cửa hàng bán trang sức

Dự án website bán trang sức với hệ thống backend và frontend.

## Cấu trúc dự án

```
aura-jewelry/
├── backend/          # Backend API
├── frontend/         # Frontend React
├── database/        # Database files
└── Screenshots/      # Hình ảnh minh họa
```

## Công nghệ sử dụng

- **Frontend**: React, Node.js
- **Backend**: Node.js, Express
- **Database**: MongoDB/SQLite

## Cách chạy

### Frontend
```bash
cd frontend
npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm start
```
